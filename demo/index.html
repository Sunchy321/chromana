Chromana

A TCG symbol font with ligature support

Available Icons

Ligature Demo

Type icon names to see the ligature effect:

circle square diamond triangle cross hourglass

Usage

1. Include CSS

<link rel="stylesheet" href="path/to/chromana.css">

2. Using CSS Classes

<i class="cm cm-circle"></i>
<i class="cm cm-square"></i>
<i class="cm cm-triangle"></i>

3. Using Ligatures

<span class="cm">circle</span>
<span class="cm">square</span>
<span class="cm">triangle</span>
