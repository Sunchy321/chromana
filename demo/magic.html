使用示例

在HTML中使用

<!-- 在CSS中引入字体 -->
<link rel="stylesheet" href="path/to/magic.css">

<!-- 使用图标 -->
<i class="magic-icon">图标代码</i>

<!-- 使用带阴影效果的图标 -->
<i class="magic-icon shadow">图标代码</i>

在CSS中使用

.my-icon::before {
  font-family: 'Chromana-magic';
  content: '图标代码';
  /* 其他样式 */
  font-size: 24px;
  color: #333;
}
