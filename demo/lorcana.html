提示: 输入符号代码来测试连字功能。启用"阴影模式"可以切换显示带阴影的符号。

使用示例

在HTML中使用

<!-- 在CSS中引入字体 -->
<link rel="stylesheet" href="path/to/lorcana.css">

<!-- 使用图标 -->
<i class="lorcana-icon">图标代码</i>

<!-- 使用带阴影效果的图标 -->
<i class="lorcana-icon shadow">图标代码</i>

在CSS中使用

.my-icon::before {
  font-family: 'Chromana-lorcana';
  content: '图标代码';
  /* 其他样式 */
  font-size: 24px;
  color: #333;
}
